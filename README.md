### 1. **裝修項目本體論圖（Renovation Project Ontology Graph）**
- **頂層分類**：
  - 住宅類型（House Type）
    - 公寓
    - 獨棟房屋
    - 別墅
  - 裝修範圍（Renovation Scope）
    - 全屋裝修
    - 局部裝修（如廚房、浴室）
    - 軟裝裝修（如牆面、地板）

- **中層分類**：
  - 房間類型（Room Type）
    - 客廳
    - 臥室
    - 廚房
    - 浴室
    - 書房
  - 裝修項目（Renovation Item）
    - 天花板
    - 牆面
    - 地板
    - 燈光設計

- **底層分類**：
  - 材料選擇（Material Choices）
    - 地板材質：PVC、木地板、瓷磚
    - 牆面材料：乳膠漆、壁紙
    - 天花板材料：石膏板、木材
  - 工具與方法（Tools and Methods）
    - 安裝工具
    - 施工方法（DIY或專業施工）
  - 品牌選擇（Brand Choices）
    - OBA地板
    - 各類乳膠漆品牌

### 2. **AI設計建議本體論圖（AI Design Recommendation Ontology Graph）**
- **頂層分類**：
  - 用戶需求（User Needs）
    - 美學需求（Aesthetic Needs）
    - 功能性需求（Functional Needs）
    - 預算需求（Budget Needs）

- **中層分類**：
  - 設計風格（Design Styles）
    - 北歐風
    - 現代風
    - 工業風
  - 預算選擇（Budget Options）
    - 高端
    - 中端
    - 經濟型

- **底層分類**：
  - 風格元素（Style Elements）
    - 配色方案（Color Schemes）
    - 材料組合（Material Combinations）
  - AI建議算法（Recommendation Algorithms）
    - 色彩匹配算法
    - 空間利用率算法
    - 成本效益算法
